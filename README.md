# Resources for Reproducible Research in Collaborative Academic Medical Center Biostatistics Units
Supplementary material for the paper "Implementing Reproducible Studies in Collaborative Academic Medical Center Biostatistics Units"

The following resources provide examples of artifacts supporting reproducibile research for biostatistic units in academic medical centers.
This is is intended to be a "Living Supplement", allowing for updates as new resources are made available, and correcting for any URL changes to ensure accuracy and relevance.
