### Contributing New Resources

If you have a new resource you would like to suggest be added to this list, please submit a pull request or create an issue.  You should provide:

* Category of resource
* Resource name
* Resource description
* Link to resource or instructions on how to access resource file to include in the repository.

While this repository was created as a "Living Supplement" to a publication, contributors to this repository are not considered co-authors or contributors to the original publication. However, you will be listed on this Contributors page.
